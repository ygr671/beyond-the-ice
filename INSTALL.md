# beyond-the-ice
Projet développé avec **Godot Engine 4.5**.  
Ce guide décrit toutes les étapes nécessaires pour installer les prérequis, cloner le projet et le lancer correctement.

---

## Prérequis

Avant de commencer, assurez-vous d’avoir installé les logiciels suivants :

1. **Godot Engine 4.5**  
   - Téléchargement officiel : [https://godotengine.org/download](https://godotengine.org/download)  
   - Installer Godot selon votre système d’exploitation (Windows, macOS, Linux)  
   - Vérifier que Godot se lance correctement après installation

2. **Git**  
   - Téléchargement officiel : [https://git-scm.com/downloads](https://git-scm.com/downloads)  
   - Installer Git selon votre système d’exploitation  
   - Vérifier l’installation en ouvrant un terminal et en tapant :  
     ```bash
     git --version
     ```
     Vous devriez voir la version installée de Git.

3. **Éditeur de code externe (optionnel)**  
   - Visual Studio Code, Sublime Text, ou tout éditeur compatible GDScript/C#  
   - Permet de modifier facilement les scripts du projet  

---

## Étapes d’installation et d’ouverture du projet

### 1. Cloner le projet

1. Ouvrez un terminal (Windows : PowerShell, macOS/Linux : Terminal)  
2. Tapez les commande suivantes pour cloner le projet depuis GitHub puis le lancer:

en https :
```bash
git clone https://github.com/ygr671/beyond-the-ice.git
```

en ssh :
```bash
git clone git@github.com:ygr671/beyond-the-ice.git
```
puis :
```bash
cd beyond-the-ice
```

sur Windows :
tapez :
```bash
beyond-the-ice.exe
```
sur Linux/MacOS :
tapez :
```bash
chmod +x beyond-the-ice
./beyond-the-ice
```
